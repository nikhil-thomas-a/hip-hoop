— Methodology

This project treats each hip-hop mention as a natural experiment — a discrete event in a player's career that we can use to measure before/after performance.

Data Sources

NBA game logs — pulled via nba_api for every player in the dataset, covering the seasons immediately surrounding each mention's release date.

Lyric metadata — fetched via the Genius API. Song release dates are used as the event date.

Sentiment scoring — VADER (Valence Aware Dictionary and sEntiment Reasoner) applied to each lyric snippet, producing a continuous score from −1 (most negative) to +1 (most positive). This is used alongside the manual compliment/diss label as an independent predictor.

Statistical Windows

The baseline is the 30 games immediately preceding the release date. Post-mention windows are: next 1 game, next 10 games (~2 weeks), next 30 games (~1 month), and the remainder of that NBA season.

Normalization

Pace adjustment — NBA pace has varied significantly across eras (1990s: ~90 possessions/game; 2020s: ~100). All stats are scaled to a 2010s reference pace so that Shaq (1994) and Giannis (2018) can be compared fairly.

Era z-scores — for the primary 30-game window, deltas are also z-scored within era groups to produce standardized effect sizes.

Statistical Tests

Paired t-tests — test whether the mean before/after difference is statistically significant across all mentions.

Mann-Whitney U — non-parametric test comparing compliment vs. diss group distributions.

Pearson correlation — measures linear association between artist tier, VADER score, and performance delta.

OLS regression — models composite delta as a function of artist tier, mention type, and sentiment score.

Limitations

This is a correlational study. We cannot claim that hip-hop mentions cause performance changes. Confounding variables include hot/cold streaks, opponent strength, injury recovery, and seasonal variation. The dataset of ~110 mentions is sufficient for exploratory analysis but underpowered for strong causal claims. We report effect sizes and p-values honestly.

Selection bias — we only know about famous mentions. Less prominent references are underrepresented.

Composite Impact Score

Weighted average of pace-adjusted 30-game deltas: PTS (40%) + AST (30%) + REB (30%), then multiplied by artist tier for the weighted version. This is a descriptive tool, not a statistical test.
